About this micro-project

This is a compact, polished component you can add to your portfolio. It demonstrates clean UI, small JavaScript logic (random selection, copy, share), accessibility considerations, and responsive layout.

You can easily extend it to fetch quotes from an API, add favorites, or save quotes to localStorage.
